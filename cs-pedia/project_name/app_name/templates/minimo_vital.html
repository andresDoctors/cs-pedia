The maximum flow problem

The goal of the maximum flow problem is to find the maximum amount of flow that can be sent from a designated source node `s` to a designated sink node `t`, while satisfying the capacity constraints on the edges. This problem has numerous practical applications, such as determining the maximum amount of traffic that can flow through a transportation network or the maximum amount of data that can be transmitted through a communication network.

The maximum flow problem can be solved using a variety of algorithms, these are some of the most famous:

Name Time complexity Space complexity
Ford-Fulkerson O(|E| * f) O(V + E)
Edmonds-Karp O(V * E**2) O(V + E)
Dinic's O(V**2 * E) O(V + E)
Relabel-to-front O(V**3) O(V + E)
